# 📘 Module 2 – Object-Oriented Programming (OOP)

**Duration:** 1 Week (Week 3)

---

# 📖 Module Overview

This module introduces the fundamentals of **Object-Oriented Programming (OOP)** in Python. Students will learn how to organize programs using classes and objects, create reusable code, and model real-world electrical engineering components using object-oriented design.

The module follows the **Corey Schafer Python OOP** playlist and focuses only on the concepts covered in the selected resource.

---

# 🎯 Learning Outcomes

By the end of this module, students will be able to:

- Understand the concept of Object-Oriented Programming.
- Differentiate between classes and objects.
- Create classes and instantiate objects.
- Use constructors (`__init__`) to initialize objects.
- Create and access instance variables.
- Work with class variables.
- Create and use class methods.
- Create and use static methods.
- Apply inheritance to reuse code.
- Understand and implement special (magic) methods.
- Use property decorators (`@property`, `@setter`, `@deleter`) to manage attributes.
