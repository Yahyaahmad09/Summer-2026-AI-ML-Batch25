# 📝 Study Guidelines

For every lesson:

- Watch the assigned lecture before reading.
- Type every code example yourself instead of copying.
- Execute every example in Jupyter Notebook.
- Modify the examples and observe the output.
- Complete the assigned reading tasks.
- Finish the weekly practice exercises.
- Write **3–5 key learning points** after every study session.

---

# 💡 Learning Tips

- Focus on understanding arrays before learning advanced operations.
- Practice indexing and slicing regularly.
- Learn broadcasting through experimentation.
- Use Electrical Engineering numerical examples such as:
  - Voltage (V)
  - Current (A)
  - Resistance (Ω)
  - Power (W)
  - Energy (kWh)
- Commit your work to GitHub after completing each practice session.

---

# 📦 Expected Outcome

After completing this module, students should be able to:

- Understand why NumPy is used.
- Create and manipulate NumPy arrays.
- Apply indexing and slicing techniques.
- Perform mathematical and statistical operations.
- Use broadcasting efficiently.
- Solve basic numerical problems using NumPy.
- Apply simple linear algebra operations.
- Prepare numerical datasets for analysis using Pandas.
