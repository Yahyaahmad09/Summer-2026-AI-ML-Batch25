
# 🐍 Module 1 – Python Programming

**Duration:** 2 Weeks

## 📖 Description

This module introduces the fundamentals of Python programming. Students will learn Python syntax, programming logic, functions, data structures, file handling, and practical programming skills required before studying data science, machine learning, and deep learning.

---

## 🎯 Learning Outcomes

By the end of this module, students will be able to:

- Write clean and readable Python programs.
- Understand variables, operators, conditions, and loops.
- Create reusable functions.
- Work with Python lists, tuples, and dictionaries.
- Read and write files.
- Build small command-line applications.
- Organize Python code into scripts and modules.
