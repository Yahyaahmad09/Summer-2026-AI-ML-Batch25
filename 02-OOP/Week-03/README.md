# 📅 Week 3 – Object-Oriented Programming (OOP)

## 🎯 Weekly Goal

By the end of this week, students will be able to:

- Understand the concept of Object-Oriented Programming (OOP).
- Create classes and objects in Python.
- Use constructors (`__init__`) to initialize objects.
- Work with instance and class variables.
- Create and use class methods and static methods.
- Apply inheritance to reuse code.
- Understand special (magic) methods.
- Use property decorators (`@property`, `@setter`, `@deleter`) to manage object attributes.
