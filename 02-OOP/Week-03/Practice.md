# 💻 Week 3 – Practice Tasks

Practice each concept immediately after watching the corresponding lectures.

---

# 📅 Monday

## Topics

- Classes
- Objects
- Constructors (`__init__`)
- Instance Variables

---

### 🟢 Level 1

Create classes for:

- Battery
- Light Bulb
- Fan

Create one object for each class and display its information.

---

### 🟡 Level 2

Create an **Electrical Appliance** class.

Each appliance should have:

- Name
- Voltage (V)
- Current (A)

Create at least three appliance objects and display their information.

---

### 🔴 Level 3

Create a **Solar Panel** class.

Each object should store:

- Panel Name
- Rated Power (W)
- Efficiency (%)
- Manufacturer

Create multiple solar panel objects and display their details.

---

# 📅 Wednesday

## Topics

- Class Variables
- Class Methods
- Static Methods
- Inheritance

---

### 🟢 Level 1

Create a **Battery** class that contains:

- A class variable for Battery Type.
- An instance variable for Voltage.

Create multiple battery objects and observe the difference between class and instance variables.

---

### 🟡 Level 2

Create an **Electrical Appliance** parent class.

Create child classes for:

- Fan
- Light Bulb
- Motor

Display the information of each object using inheritance.

---

### 🔴 Level 3

Create a **Transformer** class.

Requirements:

- Use a class variable for Frequency (50 Hz).
- Use a class method to change the manufacturer.
- Use a static method to display a safety message.

Create multiple transformer objects and test all methods.

---

# 📅 Friday

## Topics

- Special (Magic) Methods
- Property Decorators

---

### 🟢 Level 1

Modify the **Battery** class by adding:

- `__str__()`

Display the object directly using `print()`.

---

### 🟡 Level 2

Create a **Motor** class.

Use the `@property` decorator to display:

- Motor Name
- Rated Power
- Voltage

---

### 🔴 Level 3

Create an **Electrical Equipment Management System**.

The program should:

- Create multiple equipment objects.
- Display equipment information using `__str__()`.
- Update equipment information using `@property` setters.
- Delete equipment information using `@property` deleters.
