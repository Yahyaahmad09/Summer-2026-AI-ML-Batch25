# 📘 Module 3 – NumPy

**Duration:** 1 Week (Week 4)

---

# 📖 Module Overview

This module introduces **NumPy (Numerical Python)**, the fundamental library for numerical computing in Python. Students will learn how to efficiently create and manipulate arrays, perform mathematical operations, use broadcasting, generate random data, and apply basic linear algebra concepts.

The module follows the selected **FreeCodeCamp NumPy Tutorial** as the primary learning resource, with the official NumPy documentation and Real Python articles as supplementary references.

Throughout this module, students will work with **Electrical Engineering datasets and numerical examples** such as voltage, current, power, resistance, energy, and sensor measurements.

---

# 🎯 Learning Outcomes

By the end of this module, students will be able to:

- Understand why NumPy is used.
- Create and manipulate NumPy arrays.
- Understand array dimensions, shapes, and data types.
- Perform indexing and slicing operations.
- Apply broadcasting.
- Perform mathematical and statistical operations.
- Generate random numbers using NumPy.
- Apply basic linear algebra operations.
- Prepare numerical data for Pandas.
