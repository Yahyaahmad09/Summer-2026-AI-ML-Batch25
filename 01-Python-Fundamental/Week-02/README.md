# 📅 Week 2 – Python Data Structures & Practical Programming

## 🎯 Weekly Goal

By the end of this week, students will be able to:

- Work with Python lists, tuples, and dictionaries.
- Use list and dictionary comprehensions.
- Read and write text files.
- Work with CSV and JSON files.
- Organize Python code into modules and scripts.
- Understand command-line arguments using argparse.
