# 📅 Week 5 – Pandas Fundamentals

## 🎯 Weekly Goal

By the end of this week, students will be able to:

- Understand the purpose of Pandas in data analysis.
- Create and work with Series and DataFrames.
- Load data from CSV files.
- Explore and inspect datasets.
- Select rows and columns.
- Filter and sort data.
- Perform basic operations on engineering datasets.
