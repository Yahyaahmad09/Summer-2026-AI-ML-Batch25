# 📅 Week 7 – Data Visualization with Matplotlib

## 🎯 Weekly Goal

By the end of this week, students will be able to:

- Understand the importance of data visualization.
- Create different types of plots using Matplotlib.
- Customize charts using titles, labels, legends, and styles.
- Visualize data stored in NumPy arrays and Pandas DataFrames.
- Create professional engineering graphs.
- Save figures for reports and presentations.
