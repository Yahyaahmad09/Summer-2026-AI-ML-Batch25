# 📅 Week 4 – NumPy Fundamentals

## 🎯 Weekly Goal

By the end of this week, students will be able to:

- Understand why NumPy is used in scientific computing.
- Create and manipulate NumPy arrays.
- Understand array properties and dimensions.
- Perform indexing and slicing operations.
- Apply broadcasting for efficient computations.
- Perform mathematical and statistical operations.
- Work with basic linear algebra functions.
- Generate random data using NumPy.
- Apply NumPy to simple Electrical Engineering datasets.
