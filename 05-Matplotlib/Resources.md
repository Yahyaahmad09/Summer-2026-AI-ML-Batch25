# 📝 Study Guidelines

For every lesson:

- Watch the assigned lecture before reading.
- Type every code example yourself instead of copying.
- Execute every example in Jupyter Notebook.
- Modify the examples and observe the output.
- Complete the assigned reading tasks.
- Finish the weekly practice exercises.
- Write **3–5 key learning points** after every study session.
- Commit your progress to GitHub after completing each study day.

---

# 💡 Learning Tips

- Learn one chart type at a time.
- Focus on understanding **when** to use each visualization.
- Practice customizing every graph.
- Experiment with colors, markers, legends, grids, and figure sizes.
- Always label your graphs clearly.
- Build plots using the Electrical Engineering datasets created in previous modules.

# 📂 Software Requirements

Before starting this module, ensure the following packages are installed:

- Python 3.x
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook

Install Matplotlib using:

```bash
pip install matplotlib
```

or

```bash
conda install matplotlib
```
