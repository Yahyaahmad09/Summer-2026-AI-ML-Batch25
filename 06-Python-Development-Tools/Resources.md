# 📝 Study Guidelines

For every lesson:

- Watch the assigned videos.
- Follow every demonstration step-by-step.
- Install and configure the required tools.
- Practice every command yourself.
- Organize your projects professionally.
- Commit your work to GitHub regularly.

---

# 💡 Learning Tips

- Use virtual environments for every new project.
- Write meaningful commit messages.
- Keep your repositories organized.
- Use Git frequently instead of waiting until the project is finished.
- Learn keyboard shortcuts in VS Code.
- Practice debugging instead of only reading about it.
- Always maintain a clean project structure.

# ⚙️ Software Requirements

Install the following software before beginning this module:

- Python 3.x
- Visual Studio Code
- Git
- Jupyter Notebook

Install required packages:

```bash
pip install notebook
```

or

```bash
pip install jupyter
```

# 📂 Professional Project Structure

A recommended Python project structure:

```text
project/
│
├── data/
├── notebooks/
├── src/
├── images/
├── requirements.txt
├── README.md
└── .gitignore
```
