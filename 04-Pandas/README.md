# 📘 Module 4 – Pandas

**Duration:** 2 Weeks (Week 5 – Week 6)

---

# 📖 Module Overview

This module introduces **Pandas**, the most widely used Python library for data analysis and manipulation. Students will learn how to work with tabular data, clean datasets, perform analysis, and prepare engineering data for visualization.

Throughout this module, students will use **Electrical Engineering datasets** such as electrical load, voltage, current, power consumption, solar generation, and energy measurements.

The module follows the selected **Keith Galli Pandas** course together with the official Pandas documentation.

---

# 🎯 Learning Outcomes

By the end of this module, students will be able to:

- Understand Pandas Series and DataFrames.
- Read data from CSV, Excel and JSON files.
- Explore and inspect datasets.
- Select, filter and sort data.
- Handle missing values.
- Modify and clean datasets.
- Group and summarize data.
- Merge and combine datasets.
- Perform basic statistical analysis.
- Prepare engineering datasets for visualization.
