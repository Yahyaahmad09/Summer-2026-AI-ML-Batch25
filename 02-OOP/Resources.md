# 📚 Module 2 – Learning Resources

This document contains all learning resources for **Module 2 – Object-Oriented Programming (OOP)**.

Students are encouraged to follow the resources in the given order.

---

# 🎥 Primary Resource

## Corey Schafer – Python OOP Tutorials

**Description**

The primary resource for this module. The playlist introduces Object-Oriented Programming in Python through practical examples and covers the core OOP concepts used throughout the course.

**Topics Covered**

- Introduction to OOP
- Classes and Objects
- Constructors (`__init__`)
- Instance Variables
- Class Variables
- Class Methods
- Static Methods
- Inheritance
- Special (Magic) Methods
- Property Decorators (`@property`, `@setter`, `@deleter`)

**Link**

https://www.youtube.com/playlist?list=PL-osiE80TeTsqhIuOqKhwlXsIBIdSeYtc

---

# 📝 Study Guidelines

For each assigned resource:

- Watch or read the complete material.
- Follow along by typing the code instead of copying it.
- Execute every example program.
- Modify the examples and observe the output.
- Write **3–5 key learning points** in your notes.
- Save useful code snippets for future reference.

---

# 💡 Learning Tips

- Practice after every lecture instead of waiting until the end.
- Create your own classes using simple Electrical Engineering examples.
- Experiment by changing attribute values and methods.
- Focus on understanding how objects interact rather than memorizing syntax.
- Commit your work regularly to your GitHub repository.

---

# 📦 Expected Outcome

After completing these resources, students should be able to:

- Design simple Python classes.
- Create and use objects.
- Organize code using methods and attributes.
- Reuse code through inheritance.
- Build small Object-Oriented Python programs for Electrical Engineering applications.
